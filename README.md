# Monopoly

A Python based take on the all-time classic board game 'Monopoly' using the pygame module.

# Game Play
- [Install Pygame](https://www.pygame.org/wiki/GettingStarted)
- Then, clone this repository and run the following command
  ```
  python Monopoly.py
  ```
